# Jobify – Job Application 

## Overview

Jobify is a modern web-based job application platform built using React. The application allows users to browse job listings, view detailed job descriptions, and apply for jobs. It is designed to simulate a real-world job board system and demonstrate core frontend development skills such as component-based architecture, state management, routing, and API integration.

This project is ideal for showcasing practical React skills in a portfolio.

---

## Objectives

- Build a functional job listing platform
- Practice React fundamentals and advanced concepts
- Implement routing for multi-page navigation
- Integrate APIs for dynamic data
- Manage application state effectively
- Create a responsive and user-friendly interface

---

## Features

### Job Listings
- Displays a list of available jobs
- Each job includes title, company, location, and summary
- Jobs can be fetched from an API or stored locally

### Job Details
- Displays full details of a selected job
- Includes description, requirements, and company info
- Uses dynamic routing with job ID

### Job Application Form
- Users can apply for jobs through a form
- Fields include:
  - Name
  - Email
  - Resume / Cover Letter
- Includes basic form validation

### Add Job
- Allows users to create new job postings
- Updates UI dynamically without refresh

### Delete Job
- Allows removal of job listings
- Demonstrates state updates and UI re-rendering

### Navigation
- Multi-page navigation using React Router
- Pages include:
  - Home
  - Jobs
  - About
  - Job Details

### Responsive Design
- Styled using Tailwind CSS
- Works across mobile, tablet, and desktop devices
